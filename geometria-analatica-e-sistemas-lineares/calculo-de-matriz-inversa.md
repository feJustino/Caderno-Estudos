# Matriz inversa

## Matriz Identidade
- Matriz qudarada; Numero de linha = Numero de colunas; Diagonal principal tem todos elementos = 1 e todos os outros elementos = 0; representado por I;
## Matriz Inversa
- Matriz quadrada; Ocorre quando o produto de duas matrizes resulta numa matriz identidade de mesma ordem; Para encontrar a inversa de matriz utiliza-se multiplicação (A.B = B.A = In)
## Propriedades da Matriz Inversa
- Existe somente uma matriz inversa para cada matriz; A matriz transposta de uma matriz inversa também é a inversa (formula vide caderno); Nem todas matriz possui uma inversa; inversivel somente quando os produtos resultam numa matriz identidade
