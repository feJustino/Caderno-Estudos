## Fundamentos do JavaScript
- Variáveis e tipos de dados
    > Variáveis e tipos de dados são fundamentais em qualquer linguagem de programação. No JavaScript, as variáveis são definidas por palavras-chave como var, let e const, permitindo a manipulação de valores que podem ser alterados durante a execução do códig
- Operadores
- Estruturas de controle
    > Operadores permitem realizar operações matemáticas e lógicas em dados. No JavaScript, existem operadores aritméticos, de comparação e lógicos, além de estruturas de controle, como laços de repetição, que gerenciam o fluxo do código e evitam loops infinitos.
- Funções
- Objetos e arrays
    > Funções são essenciais para reutilização de código, permitindo que blocos de código sejam escritos uma vez e utilizados várias vezes. No JavaScript, existem diferentes formas de declarar funções, e objetos representam entidades do mundo real com propriedades em pares de chave-valor.
## Constantes e Variáveis
- var -> escopo global ou de função
- let -> Escopo de bloco
- const -> escopo de bloco, valor inalterável
